## 🩵진행 방식

### 시간
- 주 1회 1시간-1시간30분
### 장소
- 대면/비대면 병행
- 대면시 학교 공학관에서 진행
### 학습 방식
[스터디 시간 전]
- 각 주차에 해당하는 주제를  PPT, 노션, 블로그 등 편한 방식으로 정리해 스터디 전날까지 업로드
- 면접 예상 질문 만들기 (2-3문제)
  - 정답도 별도 파일에 작성하기
- 각자 업로드한 내용을 읽어보기
  
[스터디 시간]
- 퀴즈 풀어보고 서로 모르는 내용 공유
- 예상 면접 질문에 각자 답을 작성해보기
- 답 공유하는 시간 갖기
